◆ getBField()

HallSwitch::Result_t HallSwitch::getBField ( )

Gets magnetic field.

This functions returns the last read value updated in the object member bfieldVal variable, either from a explicit readBField() call or due to an interrupt event.

If polling mode is used, readBField() has to be called preivously in order to get the actual value.

Returns
HallSwitch magnetic field result

  • sensor output GPIO pointer

Optional arguments:

  • interrupt callback. By default NULL. Required to enable interrupt mode.
  • sensor power GPIO pointer (only for switch powered mode platform). By default it is NULL.
Parameters
[in]*outputSensor output GPIO interface pointer
[in]cBackCallback for interrupt mode. When passed, it enables interrupt mode
[in]*powerSensor switch poewr controller GPIO interface pointer. Default NULL will set power mode to MAIN

◆ readBField()

HallSwitch::Result_t HallSwitch::readBField ( )

Reads the actual magnetic field value.

  • Magnetic field present if the GPIO voltage level is low
  • Mangnetic field not present if the GPIO voltage level is high

The "bfieldVal" object member is updated with the read value.

Returns
HallSwitch magnetic field value
Return values
B_FIELD_ONif magnetic field present
B_FIELD_OFFif magnetic field NOT present
